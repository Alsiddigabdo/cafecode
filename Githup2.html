درس إعداد حساب على GitHub

الخطوات

  1. تسجيل حساب: قم بزيارة موقع GitHub وانقر على "Sign Up" لإنشاء حساب جديد باستخدام عنوان بريدك الإلكتروني.
  2. إعداد الحساب: قم بتعبئة معلومات الحساب المطلوبة واختر كلمة مرور آمنة.
  3. تأكيد البريد الإلكتروني: بعد إنشاء الحساب، قم بتفعيله من خلال الرابط الذي ستتلقاه عبر البريد الإلكتروني الذي قمت بتقديمه.
  4. إعداد Git: إذا لم يكن لديك Git مثبتًا على نظامك، يمكنك مراجعة درس تثبيت Git لتثبيته.
  5. إنشاء مفتاح SSH: من المهم ربط حساب GitHub بـ Git باستخدام مفتاح SSH. اتبع هذه الخطوات لإنشاء مفتاح SSH وإضافته إلى حسابك على GitHub.
  6. إضافة المفتاح إلى GitHub: قم بنسخ محتوى المفتاح العام الذي تم إنشاؤه، ثم قم بالانتقال إلى إعدادات حسابك على GitHub > SSH and GPG keys > New SSH key. قم بلصق المفتاح وحفظه.

البدء في الاستفادة من GitHub

بعد إعداد حسابك على GitHub وربطه بـ Git، يمكنك البدء في إنشاء مشاريعك وتحميلها إلى مستودعات GitHub للتحكم في إصداراتها والتعاون مع فريق العمل.
