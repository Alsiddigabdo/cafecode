الوسم التعريف
color يحدد لون النص.
background-color يحدد لون خلفية العنصر.
font-size يحدد حجم الخط.
font-family يحدد نوع الخط.
font-weight يحدد وزن الخط (عريض أو عادي).
text-align يحدد توجيه النص (من اليمين أو اليسار أو الوسط).
padding يحدد المسافة بين حاشية العنصر ومحتواه الداخلي.
margin يحدد المسافة بين حاشية العنصر والعناصر المجاورة.
border يحدد سمك ونوع ولون حاشية العنصر.
width يحدد عرض العنصر.
height يحدد ارتفاع العنصر.
display يحدد كيفية عرض العنصر (مثل block أو inline).
border-radius يحدد نصف قطر الزاويا لعنصر مربع أو مستطيل.
box-shadow يضيف ظل لعنصر بالاستفادة من قيم مثل اللون والتوجيه والبعد.
text-decoration يحدد تزيين النص مثل الخط السفلي أو الخط العلوي أو خط العلبة.
opacity يحدد مدى شفافية العنصر (قيمة بين 0 و1).
transition يسمح بتحديد الخصائص التي ستتغير تدريجياً عند تغير الحالة.
transform يمكن استخدامه لتحويل العنصر، مثل التدوير والتحجيم.
media يمكن استخدامه لتحديد تنسيقات الشاشة المختلفة لتطبيق تنسيقات مختلفة.
@keyframes يمكن استخدامه لإنشاء تحركات ورسوم متحركة باستخدام الإطارات الرئيسية.
@media يمكن استخدامه لتطبيق تنسيقات مختلفة بناءً على نطاق الشاشة.
@import يستخدم لاستيراد ملف CSS آخر داخل ملف CSS حالي.
flexbox نظام لترتيب وتوزيع العناصر في صفحة بطريقة مرنة وسهلة.
grid نظام لإنشاء تخطيط شبكي مرن ومتقدم يتيح توزيع العناصر على الصفوف والأعمدة.
nth-child يحدد العناصر بناءً على ترتيبها في الوالد الأم (العنصر الأب).
before و after يمكن استخدامهما لإضافة محتوى إضافي قبل أو بعد عنصر معين.
box-sizing يحدد كيفية حساب عرض وارتفاع العنصر (مع حاشية وحدود أم لا).
outline يحدد حاشية مؤقتة حول العنصر بدلاً من الحدود.
resize يسمح للعنصر بتغيير حجمه بواسطة المستخدم.
user-select يتحكم في إمكانية تحديد نصوص الصفحة من قبل المستخدم.
