الخلفيات والأشكال (Backgrounds and Shapes)

الخلفيات والأشكال من العناصر المهمة في تصميم صفحات الويب. باستخدام قواعد CSS3، يمكنك تحديد الخلفيات بأنواعها المختلفة وإنشاء أشكال هندسية متنوعة.

تحديد خلفيات الصفحة والعناصر

مثال على تحديد خلفية للصفحة ولعنصر:


  /* تحديد خلفية للصفحة */
  body {
    background-image: url('background.jpg');
    background-size: cover;
    background-repeat: no-repeat;
    background-attachment: fixed;
  }
  
  /* تحديد خلفية لعنصر */
  .box {
    width: 200px;
    height: 200px;
    background-color: #f4f4f4;
    background-image: linear-gradient(to bottom, #f4f4f4, #cccccc);
    border-radius: 50%;
  }
  
  /* تحديد شكل هندسي */
  .triangle {
    width: 0;
    height: 0;
    border-left: 50px solid transparent;
    border-right: 50px solid transparent;
    border-bottom: 100px solid #ff6600;
  }

الاستنتاج

تحديد الخلفيات وإنشاء الأشكال باستخدام CSS3 يمكن أن يُضفي لمسة جمالية وإبداعية على تصميمات صفحات الويب. باستخدام هذه القواعد، يمكنك تحقيق تأثيرات خلفيات مختلفة وإنشاء أشكال هندسية متعددة.
